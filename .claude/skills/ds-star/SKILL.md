# ds_star - Decision Support

**Location:** `/Volumes/LegacySafe/SS_III/ds_star/`

## What It Does

AI-powered analysis and strategy building:

- **SynopticCore** - Smart Asset Score (0-100 rating system)
- **OracleInterface** - Natural language to chart analysis
- **ArchitectForge** - Natural language to strategy builder
- **TransparentAnalyst** - Step-by-step reasoning engine

## Key Modules

```
SynopticCore        - Asset scoring engine
OracleInterface     - NL → Chart conversion
ArchitectForge      - NL → Strategy generation
TransparentAnalyst  - Explainable reasoning
```

## How to Use

```bash
cd /Volumes/LegacySafe/SS_III/ds_star

# Score an asset
python SynopticCore.py --asset BTC

# Generate strategy from natural language
python ArchitectForge.py --prompt "Buy BTC on RSI oversold"

# Analyze with explanation
python TransparentAnalyst.py --query "Why is SOL trending?"
```
